### Cronômetro
Projeto para práticar [vueJS](https://cli.vuejs.org/), um cronômetro simples com javascript, css e html.
### Get started
```
# Instale as dependência:
$ npm install

# Para desenvolvimento:
$ npm run serve

# Para produção:
$ npm run build
```
